# ✈️Flight Reservation System in C

A console-based flight reservation system developed using C programming and file handling concepts.  
The system allows users to book flights while providing an admin panel to manage flight records.

---

## Features
User flight booking  
Admin login system  
Add new flights  
Automatic seat updates  
Passenger data stored in files  
Structured and modular code  

---

## Technologies Used
- C Programming
- File Handling
- Structures
- Modular Functions

---

## How to Run

1. Clone the repository:
git clone https://github.com/Divya-n-reddy/flight-reservation-system-c.git

2. Navigate to the project folder:
cd flight-reservation-system-c

3. Compile:
gcc main.c -o flight

4. Run:
./flight
